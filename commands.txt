Common conda Commands
Environment Management:
conda create -n <env_name> <package_list>: Creates a new environment named <env_name> and installs specified packages.
conda activate <env_name>: Activates the specified environment.
conda deactivate: Deactivates the current active environment.
conda env list: Lists all available environments.
conda env remove -n <env_name>: Removes the specified environment.
Package Management:
conda install <package_name>: Installs a package into the active environment.
conda update <package_name>: Updates a specific package.
conda update --all: Updates all packages in the active environment.
conda remove <package_name>: Removes a package from the active environment.
conda list: Lists all installed packages in the active environment.
Other Useful Commands:
conda info: Provides information about the current conda installation.
conda config --show: Displays current conda configuration settings.
conda config --set <key> <value>: Sets a specific configuration value.
conda clean --all: Removes unused packages and caches.
Remember:
To install a specific version of a package, use the == operator: conda install <package_name>=<version>
To specify a channel for a package, use the -c flag: conda install -c <channel_name> <package_name>
For more advanced usage and troubleshooting, refer to the official conda documentation.
